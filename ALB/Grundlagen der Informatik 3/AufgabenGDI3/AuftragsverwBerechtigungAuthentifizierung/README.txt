Um die Applikation AVApplikation ausführen und testen zu können, 
führen Sie bitte folgende Schritte durch:

1. Führen Sie die Schritte im Wissensbaustein "Kurzanleitung: Java DB***" 
   zum Einrichten eines Datenbankservers durch.
   Für das Lesezeichen wählen Sie 
   Userid: Benutzer
   Passoword: Kennwort
   JDBC URL: jdbc:derby://localhost:1527/Auftragsverwaltung;create=true

2. Verbinden Sie sich mit der Datenbank durch ein Doppelklick 
   auf das Lesezeichen.

3. Kopieren Sie die Create-Anweisungen aus dem Wissensbaustein:
   "Fallstudie: AVPlus - OOP der Datenhaltungsschicht**"
   unter "Struktur der relationalen Datenbank"
   Führen Sie die Befehle mit einem Klick auf das grüne Dreieck aus.

4. Installieren Sie als Nächstes JGoodies: Im Wissensbaustein 
   "Fallstudie: AVplus - Architektur und Komponenten**" 
   finden Sie ein Link auf die Internetseite davon. Gehen Sie dort
   auf Downloads, dann auf Libraries, dann wählen Sie die Bibliothek
   "JGoodies Forms" aus.
   Nach dem Entpacken erweitern Sie Ihr CLASSPATH, dass es auf
   forms-x.x.x.jar zeigt (Versionsnummer anpassen, wenn nötig).

5. Starten Sie die Applikation AVApplication im Paket auftragsverwaltung\gui.
